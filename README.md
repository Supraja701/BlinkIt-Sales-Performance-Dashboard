# BlinkIt-Sales-Performance-Dashboard
🛒 Blinkit Sales & Outlet Performance Dashboard

An interactive Business Intelligence dashboard built to analyze Blinkit’s sales performance, outlet distribution, item segmentation, and customer ratings.
This project focuses on transforming raw retail data into actionable business insights using data visualization techniques.

📌 Project Overview

This dashboard provides a comprehensive analysis of:

📊 Total Sales Performance

⭐ Average Ratings

🏪 Outlet Establishment Trends

🥫 Item Type Distribution

🧈 Fat Content Analysis

📍 Outlet Location & Tier Performance

🏬 Outlet Type Comparison

The goal of this project is to simulate real-world retail analytics and enable data-driven decision-making.
